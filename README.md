# Curso-Desarrollo-Web-Full-Stack

##Instalaciones Necesarias

1. Visual Studio Code
2. Postman
3. Node Js
4. TypeScript
   <pre>
     <code>npm install -g typescript</code>
   </pre>
6. Angular CLI
   <pre>
     <code>npm install -g @angular/cli</code>
   </pre>
8. Mongo Compass
